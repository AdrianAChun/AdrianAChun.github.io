Game Rules



1. The game is played on a grid that's 3 squares by 3 squares.

2. Select X or O, your friend (or the computer) will be another one. Players take turns putting their marks in empty squares.

3. The first player to get 3 of her marks in a row (up, down, across, or diagonally) is the winner.

4. When all 9 squares are full, the game is over. If no player has 3 marks in a row, the game will restart automaticlly.


Type " 1 " and " Enter " to enter a game with computer

Type " 2 " and " Enter " to enter a game with a friend

Type " 3 " and " Enter " to check your how many games you played!

Type " 4 " and " Enter " to exit the game
